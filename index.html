Aurora ISA

Aurora is a simple RISC-based instruction set that ohnx is developing in order to learn about ISA's and FPGA's.

Multiple implementations are planned, including a desktop application (C), web-based (javascript), and hardware-based (Verilog).

Aurora is designed to have a special feature, which is planned to be revealed sometime in the development process.

For info about Aurora and specifications of the ISA, please see the latest spec sheet.
